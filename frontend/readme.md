# MyReview
<b>MyReview</b>는 책/영화 리뷰를 작성할 수 있는 애플리케이션입니다. <br/>
컨텐츠 검색과 리뷰 작성 뿐 아니라 올 해의 목표를 설정하고 달성률을 보여주는 기능도 제공하여 이용자에게 동기를 부여하고자 했습니다.
